# vue-app-starter

Vue.js v3 を用いた web アプリ開発のための個人用スターター

## Description

- webpack によるバンドル
- ESlint，stylelint による静的コード解析
- Prettier によるフォーマット
- Babel によるトランスコンパイル

## Note

[eslint-plugin-vue](https://github.com/vuejs/eslint-plugin-vue) は最新バージョンを使用しVue3準拠としている．

## Usage

Install package

```
npm install
```

build

```
npm run build
```

dev

```
npm start
```
